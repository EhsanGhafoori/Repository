# Git and GitHub Final Project

A comprehensive project repository demonstrating Git and GitHub best practices, including proper documentation, licensing, and contribution guidelines.

## Description

This repository serves as a demonstration of professional repository setup and management, following industry standards for open-source projects. It includes essential files such as license documentation, code of conduct, contributing guidelines, and a sample shell script implementation.

## Features

- Apache 2.0 License for open-source distribution
- Comprehensive README documentation
- Code of Conduct for community guidelines
- Contributing guidelines for contributors
- Sample shell script (simple-interest.sh) for calculating simple interest

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```

2. Make the shell script executable:
   ```bash
   chmod +x simple-interest.sh
   ```

## Usage

### Running the Simple Interest Calculator

Execute the shell script to calculate simple interest:

```bash
./simple-interest.sh
```

The script will prompt you for:
- Principal amount
- Rate of interest (as a percentage)
- Time period (in years)
